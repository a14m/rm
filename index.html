rm

since you've reached this page,I assume you did the same bad thing I did before.

after using rm very_important_file you can't figure out how to retrieve the file. I'm not sure how to retrive that file, but I came to write this to avoid doing the same stupid mistake ever again.

you can thank me later :D

Install

sudo cp rm.sh /usr/local/bin/rm

Uninstall

sudo rm /user/local/bin/rm you might need to restart the terminal or run source ~/.bashrc or source ~/.bash_profile

Usage

rm [OPTION]...FILE...

Options

option desc.
-h,--help display this help text and exit.
--version output version information and exit.
-r, -R, --recursive move the directories and their content to the Trash folder.
-f, --forever remove the directories and their content to the Trash folder ( what you did :D ).
